# Pong-Game
Recreation of the classic game of Pong

## Table of Contents
* General Info
* Technologies
* Setup 

## General Info
This project is the game of Pong with a Tron-style theme

## Technologies
Project is created with:
* Python 3.6
* Pygame 1.9

## Setup
To run this project, install Python 2.7 or 3.6, as well as Pygame 1.9 and run.
